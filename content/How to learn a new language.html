How to learn a new language

學習新程式語言的方法, 就是先從其官方網頁下手, 了解其起源, 特性, 並結合機械設計流程專業理論分析, 學習如何用此一語言來協助解決產品開發問題.

Dart: https://dart.dev/guides

配合 Quick Start Guide to Dart Programming (2020) 與 Beginning App Development with Flutter (2019) 電子書章節說明

學習目標: 能夠利用 Flutter 編寫機械設計流程中的手機與平板前端介面程式

Python: https://www.python.org/downloads/

Building REST APIs with Flask

https://link.springer.com/book/10.1007/978-1-4842-5022-8

Applied Reinforcement Learning with Python

https://link.springer.com/book/10.1007/978-1-4842-5127-0

學習目標: 能夠利用 Python 編寫機械設計流程中的後端演化分析程式

C:

Beginning C for Arduino

https://link.springer.com/book/10.1007/978-1-4302-4777-7

學習目標:能夠利用 C 控制機協設計流程中的感測與致動裝置

C++:  

Beginning C++ 17

https://link.springer.com/book/10.1007/978-1-4842-3366-5
