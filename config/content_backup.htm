猜數字遊戲程式

http://mde.tw/cp2019/content/Task3.html

Dart  Random 類別:

import 'dart:math' as math;

var now = DateTime.now();
math.Random rand = math.Random(now.millisecondsSinceEpoch);
int answer;

void main() {
  for (int i = 0; i < 10; i++) {
    // random integer from 1 to 100
    answer = 1 + rand.nextInt(100 - 1);
    print(answer);
  }
}

猜數字程式設計流程:

設法以亂數產生介於 1~100 之間的整數, 作為答案.

利用 HTML input 表單, 接受使用者輸入所猜的整數

若所猜的數字大於答案, 回覆所猜的整數太大

使用者繼續猜

若所猜的數字小於答案, 回覆所猜的整數太小

讓使用者繼續猜, 直到猜對為止.
